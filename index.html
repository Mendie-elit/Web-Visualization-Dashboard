Project Summary





  This project examines the relationship of weather patterns in cities based on their coorelation to the equator. Using Python libraries, citipy, and API's, statistics were obtained for the max temperature, humidity, cloudiness, and wind speed for more than 500 randomly selected cities. This data was then plotted using MatPlotLib to obtain a visualization of the results.

This site contains a page for each of the four weather statistics selected, each page includes a graph and a summary of the results which may be accessed by clicking on the graph below or through the Plot menu. The Comparison page contains all four graphs and the Data page includes a table of the data for each city.
